inlinestatic

Retrieves the TaskData for this provider.

This function attempts to retrieve data from either the thread-local store or the global store, depending on the current thread context. If the data is not found or the provider ID does not match, it returns nullptr.

Parameters
provider_idThe ID of the provider whose data is to be retrieved.
Returns
A shared pointer to a constant object of type T if found, otherwise nullptr.
