Detailed Description

This namespace holds all of the communication primitives that are used by the Behaviour header to send messages to the behaviour system.

You shouldn't be emitting any of these messages outside of the behaviour header.

Provide providers are typical providers that run when given a task.

Start providers run once when a provider group first gets a task.

Stop providers run once there are no tasks running for any provider of this group.
