Detailed Description

This class is used to convert images from a permuted mosaic form to a regular mosaic form.

The use case for this is to make bayer mosaic patterns more compressable for format like JPEG by removing the hard edges that occur on colour transitions in a mosaiced image. For example it would be able to perform the conversion between the following image permutations.

RGRGRGRGRGRGRGRGRGRG RRRRRRRRRRGGGGGGGGGG GBGBGBGBGBGBGBGBGBGB RRRRRRRRRRGGGGGGGGGG RGRGRGRGRGRGRGRGRGRG RRRRRRRRRRGGGGGGGGGG GBGBGBGBGBGBGBGBGBGB RRRRRRRRRRGGGGGGGGGG RGRGRGRGRGRGRGRGRGRG -> RRRRRRRRRRGGGGGGGGGG GBGBGBGBGBGBGBGBGBGB GGGGGGGGGGBBBBBBBBBB RGRGRGRGRGRGRGRGRGRG GGGGGGGGGGBBBBBBBBBB GBGBGBGBGBGBGBGBGBGB GGGGGGGGGGBBBBBBBBBB RGRGRGRGRGRGRGRGRGRG GGGGGGGGGGBBBBBBBBBB GBGBGBGBGBGBGBGBGBGB GGGGGGGGGGBBBBBBBBBB

Each of the four components in the bayer are spread out into a distinct corner of the image.
