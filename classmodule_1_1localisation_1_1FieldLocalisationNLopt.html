void module::localisation::FieldLocalisationNLopt::uncertainty_reset ( const FieldDescription &  fd,

const Eigen::Isometry3d &  Hrw

Determines a new state by running a grid search and checking the cost of each hypothesis.

First, a local search is performed around the last certain state. If this does not find a low cost hypothesis, a global search is performed on the half of the field that the robot is currently on, to reduce computation and avoid the mirror field problem.

Parameters
fdThe field description containing the field dimensions, in particular the field length and width.
field_linesField lines, used to find the cost of hypotheses.
field_intersectionsField intersections, used to find the cost of hypotheses.
goalsGoals, used to find the cost of hypotheses.
HrwThe homogenous transformation from world {w} to robot {r} space.
